\sec Aktuální verze RTPS protokolu

* Structure Module - základní objekty
\begitems
* GUID, Locator
* přejmenování
\enditems

* Messages Module - specifikace zpráv
\begitems
* odebrány VAR, ISSUE
* přidána DATA
\enditems

* Behavior Module - chování ``Endpoints''
\begitems
* stateless
* stateful
\enditems

* Discovery Module - objevování ``Participants'', ``Endpoints''
\begitems
* SPDP, SEDP
\enditems
